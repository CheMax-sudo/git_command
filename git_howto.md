# Подсказка по GIT.
Инициализация/создание репозитория
```sh
git init 
```
Добавим все файлы проекта в нам будующий commit
```sh
git add
git add --all //Добавить все файлы.
git add .\git_howto.md //Добавление конкретного файла.
```
git commit
git status
git log
git log --oneline
git checkout
git checkout master
git branch
git merge
