# Подсказка по GIT.
Инициализация/создание репозитория
```sh
git init 
```
Добавить все файлы проекта в нам будующий commit.
```sh
git add
git add --all //Добавить все файлы.
git add .\git_howto.md //Добавление конкретного файла.
```
Добавить commit.
```sh
git commit
```

```sh
git status
```
git log
git log --oneline
git checkout
git checkout master
git branch
git merge
